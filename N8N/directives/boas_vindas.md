# Diretriz: Boas-vindas ao Usuário

Esta diretriz define como o agente deve saudar o usuário ao iniciar o uso do DOE Framework.

## Objetivo
Demonstrar o funcionamento do sistema de 3 camadas.

## Entradas (Inputs)
- Nome do usuário (opcional).

## Ferramentas de Execução
- `execution/saudacao.py`

## Saída (Output)
- Mensagem de boas-vindas formatada no terminal.
