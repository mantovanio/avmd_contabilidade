================================================================================
            COMO USAR O CRIADOR DE SKILLS (PASSO A PASSO)
================================================================================

Este guia ensina como você pode me pedir para criar novas "Habilidades" (Skills) 
usando o sistema que acabamos de configurar.

--------------------------------------------------------------------------------
1. O QUE É UMA SKILL?
--------------------------------------------------------------------------------
Uma Skill é um conjunto de regras e ferramentas que me ensinam a fazer algo 
específico de forma profissional (ex: revisar textos, organizar arquivos, 
analisar dados de planilhas, etc).

--------------------------------------------------------------------------------
2. COMO PEDIR PARA CRIAR UMA NOVA SKILL
--------------------------------------------------------------------------------
Basta falar comigo naturalmente! Use frases como:

- "Antigravity, crie uma skill para organizar meus recibos de contabilidade."
- "Crie uma skill que me ajude a escrever e-mails de vendas."
- "Gostaria de uma skill para analisar erros em arquivos de log."

Eu cuidarei de todo o trabalho técnico de criar as pastas e manuais.

--------------------------------------------------------------------------------
3. ONDE AS SKILLS FICAM GUARDADAS?
--------------------------------------------------------------------------------
Todas as suas habilidades ficam na pasta:
📂 .agents/skills/

Cada uma terá sua própria pastinha com o manual de instruções dentro.

--------------------------------------------------------------------------------
4. O QUE ACONTECE DEPOIS QUE EU CRIO A SKILL?
--------------------------------------------------------------------------------
Assim que eu terminar de criar, eu vou te avisar. A partir desse momento, 
sempre que você mencionar aquele assunto, eu já saberei usar as regras 
daquela Skill automaticamente.

--------------------------------------------------------------------------------
5. DICA DE OURO: O FRAMEWORK DOE
--------------------------------------------------------------------------------
Lembre-se que eu uso o sistema DOE:
D - Directive (O Manual: O que fazer)
O - Orchestration (Eu: Tomando as decisões)
E - Execution (Os Scripts: Fazendo o trabalho pesado)

Se precisar mudar algo em como uma Skill funciona, é só me pedir para 
"atualizar a diretriz da skill tal".

================================================================================
Precisa de ajuda agora? É só me chamar!
