# Referenciando Nós (n8n)

## Quando usar esta skill
- Para verificar as opções de autenticação de um nó específico.
- Para entender os modos de operação do nó Merge.
- Para configurar webhooks de forma correta.

## Fluxo de Trabalho (Workflow)
- [ ] Pesquisar o nó desejado na biblioteca oficial.
- [ ] Verificar os parâmetros obrigatórios.
- [ ] Validar o tipo de retorno esperado pelo nó.

## Instruções
- **HTTP Request**: Sempre verifique o "Body Parameters" vs "Query Parameters".
- **Merge**: Escolha entre "Append", "Combine", "Multiplex" ou "Wait" dependendo da necessidade.
